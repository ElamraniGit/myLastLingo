🔄 مسار استخراج الترجمة بعد الإصلاح

المستخدم يلصق رابط YouTube
        ↓
POST /api/v1/videos/process
  └─ yt-dlp --dump-json → metadata
        ↓
POST /api/v1/transcripts/extract/{video_id}
  └─ _fetch_youtube_captions(youtube_id, "en")
       ├─ yt-dlp --write-subs --sub-format vtt    (manual subs)
       ├─ yt-dlp --write-auto-subs --sub-format vtt (auto-generated)
       ├─ finds .vtt file
       ├─ _parse_vtt() → strips <timestamp><c> tags
       └─ returns segments[]
  
  إذا فشل YouTube captions:
  └─ _transcribe_with_whisper_audio_only()
       ├─ yt-dlp --extract-audio --audio-format wav
       ├─ WhisperService.transcribe(wav)
       └─ save_transcript() → DB
